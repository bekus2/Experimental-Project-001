# 🌊 ВайбКод — форум для вайбкодеров

Красивый форум-сообщество для тех, кто ловит вайб в коде. Тёмная неоновая
тема с аврора-фоном и glassmorphism, живые AJAX-взаимодействия без перезагрузок.

> Стек: **HTML5 · CSS3 · JavaScript · jQuery · AJAX · PHP 8.3 · MySQL**

## ✨ Возможности

- **Регистрация и вход** через AJAX, пароли — `password_hash` (bcrypt), защита CSRF.
- **Разделы и темы** с пагинацией, закреплением и блокировкой.
- **Сообщения** с лёгкой разметкой: `**жирный**`, `*курсив*`, `` `код` `` и ` ```блоки``` `, автоссылки.
- **Ответы без перезагрузки** — новый пост прилетает по AJAX и плавно появляется.
- **Лайки** с мгновенным AJAX-тогглом и анимацией.
- **Живой поиск** по темам и сообщениям (debounce + AJAX).
- **Профили** со статистикой (темы, сообщения, лайки).
- Полностью **адаптивный** интерфейс.

## 📁 Структура

```
config/config.php      — настройки (БД, сайт, безопасность)
sql/schema.sql         — схема БД + категории
install.php            — установщик (создаёт БД, демо-данные)
includes/              — db, функции, авторизация, шаблоны, рендер
api/                   — AJAX-эндпоинты (register, login, like, search…)
assets/css/style.css   — тема оформления
assets/js/app.js       — клиентская логика (jQuery + AJAX)
index.php, topic.php … — страницы форума
```

## 🚀 Установка

### 1. Требования
- PHP **8.3+** с расширениями `pdo_mysql`, `mbstring`
- MySQL **8.x** (или MariaDB 10.4+)

### 2. Настройка БД
Отредактируйте `config/config.php` или задайте переменные окружения:

```bash
export DB_HOST=127.0.0.1
export DB_NAME=vibecode_forum
export DB_USER=root
export DB_PASS=secret
```

### 3. Запуск установщика
Создаст базу, таблицы, демо-пользователей и стартовые темы:

```bash
php install.php
```

### 4. Старт сервера для разработки

```bash
php -S localhost:8000
```

Откройте <http://localhost:8000/>.

> ⚠️ После установки **удалите `install.php`** на боевом сервере.

## 🔒 Безопасность

- Подготовленные запросы (PDO) — защита от SQL-инъекций.
- CSRF-токены на всех изменяющих запросах.
- Экранирование вывода (`htmlspecialchars`), безопасный рендер разметки.
- `httponly` + `SameSite=Lax` для сессионных cookie.

## 🚢 Production

Направьте корень домена на каталог проекта. Для Apache при необходимости
закройте служебные каталоги; для nginx запретите прямой доступ к `config/`,
`includes/`, `sql/` и удалите `install.php`.
